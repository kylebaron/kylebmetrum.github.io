Time varying KA

Kyle Baron

Mon Feb 15 21:50:56 2016

library(mrgsolve)

About this example

  • We must initialize TAD in $GLOBAL so that it has some value from the get-go
  • EVID and TIME are available to check; TIME is the dataset time
  • SOLVERTIME is the according to the solver (equivalent to T in NONMEM)
  • Use TIME in $MAIN and $TABLE and use SOLVERTIME in $ODE

The code

code <- '
$PARAM CL=1, VC=20 
KAMAX = 1.1, KAT50 = 12, gam=5.1

$CMT GUT CENT

$GLOBAL double TAD = 0;

$MAIN if(EVID==1) TAD = TIME;

$ODE 

double KAT  = SOLVERTIME - TAD;
double KA = KAMAX*pow(KAT,gam)/(pow(KAT50,gam) + pow(KAT,gam));

dxdt_GUT = -KA*GUT;
dxdt_CENT = KA*GUT - (CL/VC)*CENT;

$CAPTURE KA
'
mod <- mread(code=code, model="timeka")
out <- mod %>% ev(amt=1000, ii=72, addl=3) %>% mrgsim(end=240, delta=0.1)
plot(out)

out %>% plot(KA~.)
